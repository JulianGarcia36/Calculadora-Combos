# Calculadora de Combos

Herramienta interna para armar y cotizar combos de plataformas de streaming, generar el texto de cotización para enviar al cliente, y llevar un historial de precios y combos guardados.

## Uso
Abre el sitio publicado en GitHub Pages, marca las plataformas que quieres combinar, ajusta dispositivos/perfiles si aplica, y copia el texto generado para enviarlo al cliente.

## Datos
Los precios, combos guardados e historial se almacenan únicamente en el navegador de cada persona (no hay sincronización entre dispositivos). Usa el botón **"Exportar respaldo"** dentro de la página para descargar una copia, y **"Importar respaldo"** para restaurarla en otro navegador o dispositivo.

## Archivos
- `index.html` — estructura de la página
- `style.css` — estilos
- `script.js` — lógica de la calculadora
- `robots.txt` — evita que buscadores indexen el sitio
